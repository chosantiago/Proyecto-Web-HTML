# Proyecto Web HTML – Práctica de Desarrollo

Proyecto personal desarrollado para practicar fundamentos de desarrollo web utilizando HTML y CSS.

## Tecnologías utilizadas

- HTML
- CSS
- Diseño responsive (media queries)

## Descripción

Sitio web multipágina con navegación entre secciones, uso de imágenes y estilos separados mediante archivos CSS.

El objetivo del proyecto fue practicar:

- Estructura de páginas web
- Organización de contenido
- Estilizado con CSS
- Adaptación básica a distintos tamaños de pantalla
